Publish Tutorial

  1. Open your ftp tool, I use Cyberduck, select SFTP from the drop down, enter the desired IP address, then enter your username and password

  2. Upload a single file by locating the file(s) you want to publish,

    3. and copying over a single file over from file explorer to your ftp,
    or copying over an entire directory by copying over the folder.
  3. You may access the index for your website(s) by opening a browser and typing the following into the search bar: "icarus.cs.weber.edu/~" + username

  5. To access a page saved withing a secondary folder, add the name of the folder and then the name of the webpage:

  6. If the folder has an index or homepage for your website(s), then you only need to type in the folder path and can leave out the name of the index/homepage!
