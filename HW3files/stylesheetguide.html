Adding CSS to a Webpage

CSS Box Model

The model displays the four main aspects of a CSS Box:

  • Content
  • Padding
  • Border
  • Margin

The Content is the image or text being put onto the webpage and has a certain width and height tied to it.

The Padding is the space between the Content and the Border and is used to ensure the content is sufficiently legible within the Border.

The Border is the box encasing the Content and aids in helping the Content pop out and grab the attention of the audience.

The Margin is the space between the Border and the edge of the Webpage, ensuring the content does not run outside of the browser viewport.
